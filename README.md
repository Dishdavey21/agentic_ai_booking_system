A student project demonstrating an **Agentic AI Orchestrator–Worker** architecture for travel logistics.## What this repo contains
- Frontend: Vite + React + TypeScript
- Core concept: Orchestrator delegates to worker “agents” (Budget / Premium / Analyst) and enforces a human approval gate before any sensitive action (e.g., checkout redirect).

## Run locally
```bash
npm install
npm run dev
npm run build
npm run preview
This project will be deployed to GitHub Pages under:
